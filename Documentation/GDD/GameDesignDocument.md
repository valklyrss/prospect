# Anime Open World Game - Game Design Document

## 1. 游戏概述

### 1.1 基本信息
- **游戏名称**: PROSPECT (代号)
- **类型**: 开放世界动作冒险 (Anime GTA)
- **目标平台**: PC (Steam), PlayStation 5, Xbox Series X/S
- **玩家数量**: 1人 (主线) + 在线多人模式 (后续更新)
- **预计开发周期**: 24-36个月
- **评级目标**: T for Teen (ESRB) / 16+ (PEGI)

### 1.2 核心概念
一款融合二次元动漫美学与开放世界犯罪题材的动作冒险游戏。玩家在一个充满活力的现代都市中，扮演一名拥有特殊能力的年轻主角，在合法与灰色地带之间游走，建立自己的人脉网络，揭开城市背后的秘密。

**一句话描述**: "在霓虹闪烁的动漫都市中，用异能书写你的传奇。"

### 1.3 独特卖点 (USP)
1. **动漫渲染风格**: 独特的 cel-shading + 后处理管线，打造电影级动漫视觉效果
2. **异能战斗系统**: 超能力与传统枪械结合，创造多样化战斗体验
3. **动态关系网络**: NPC 关系系统，你的行为会影响整个城市的势力格局
4. **载具定制**: 深度车辆改装系统，从外观到性能全面定制
5. **多线叙事**: 主线 + 多个角色支线 + 随机事件，每次游玩体验不同

## 2. 游戏世界观

### 2.1 设定概述
**时间**: 近未来 20XX年
**地点**: 虚构城市 "新海市" (Neo-Ocean City) - 融合东京、香港、新加坡元素的亚太都市

### 2.2 城市分区
| 区域 | 风格 | 主要活动 | 危险等级 |
|------|------|----------|----------|
| 中央商务区 | 摩天大楼、霓虹广告 | 商业任务、企业间谍 | ★★☆ |
| 旧城区 | 狭窄街道、传统市场 | 地下交易、情报收集 | ★★★ |
| 滨海娱乐区 | 海滩、夜店、游轮 | 社交活动、娱乐场所 | ★★☆ |
| 工业区 | 工厂、仓库、码头 | 走私、抢劫、车辆改装 | ★★★★ |
| 高科技园区 | 未来建筑、无人机 | 黑客任务、科技盗窃 | ★★★ |
| 山区豪宅区 | 别墅、私人会所 | 高端社交、潜入任务 | ★★☆ |
| 地下世界 | 地铁、下水道、隧道 | 黑市、逃亡、秘密基地 | ★★★★★ |

### 2.3 势力集团
1. **新海市警署 (NPPD)**: 维持秩序的警察力量，腐败与正义并存
2. **红龙会**: 传统黑帮，控制地下赌场和毒品交易
3. **天网科技**: 巨型科技公司，暗中进行非法实验
4. **自由佣兵联盟**: 中立武装组织，承接各种灰色任务
5. **幽灵黑客组织**: 网络恐怖分子，对抗企业垄断
6. **玩家帮派**: 由玩家建立和扩张的自定义势力

## 3. 核心玩法

### 3.1 玩法循环
```
探索城市 → 接取任务 → 准备装备/载具 → 执行任务 → 获得奖励 → 升级能力/基地 → 影响势力格局 → 解锁新内容 → 探索城市
```

### 3.2 主要玩法系统

#### 3.2.1 移动与探索
- **基础移动**: 奔跑、跳跃、攀爬、滑行
- **异能移动**: 
  - 瞬移冲刺 (短距离闪现)
  - 墙面奔跑 (沿建筑物垂直移动)
  - 滑翔翼 (从高处滑翔)
  - 载具召唤 (随时召唤个人载具)

#### 3.2.2 战斗系统
**双模式切换**:
- **普通模式**: 枪械、近战、掩体射击
- **异能模式**: 消耗能量条释放特殊能力

**异能类型 (可选/可升级)**:
| 异能 | 效果 | 战斗应用 |
|------|------|----------|
| 时间缓滞 | 减慢周围时间 | 闪避、精准射击 |
| 念力 | 移动物体 | 投掷环境物品、制造掩体 |
| 闪电链 | 释放电流 | 群体伤害、瘫痪载具 |
| 幻影分身 | 创造分身 | 诱饵、多角度攻击 |
| 能量护盾 | 防护罩 | 抵挡伤害、反弹子弹 |

#### 3.2.3 载具系统
- **类型**: 跑车、摩托车、越野车、直升机、快艇、未来载具
- **改装**: 引擎、悬挂、装甲、武器、涂装、霓虹灯
- **物理**: 真实驾驶手感 + 特技动作 (漂移、跳跃、墙面驾驶)

#### 3.2.4 潜行与黑客
- **潜行**: 掩体系统、无声处决、干扰装置
- **黑客**: 入侵摄像头、控制电子设备、窃取数据、制造混乱

### 3.3 任务类型

#### 主线任务 (约 20-25小时)
- 章节式推进，揭示城市阴谋
- 关键决策影响结局走向 (多结局)

#### 支线任务
| 类型 | 描述 | 示例 |
|------|------|------|
| 势力任务 | 为特定集团完成任务 | 为红龙会护送货物 |
| 角色任务 | 深化与特定NPC关系 | 帮助黑客朋友复仇 |
| 随机事件 | 城市中随机触发 | 街头抢劫、车祸救援 |
| 挑战任务 | 测试玩家技能 | 限时赛车、格斗锦标赛 |
| 收集任务 | 探索收集要素 | 寻找隐藏数据芯片 |

#### 自由活动
- 赛车比赛 (街头/合法赛道)
- 格斗竞技场
- 赌场小游戏
- 摄影模式 (城市风景)
- 房地产投资
- 股票交易 (游戏内)

## 4. 角色系统

### 4.1 主角
**默认设定** (可自定义):
- 年龄: 22岁
- 背景: 退役军人/实验体逃脱/街头孤儿 (可选)
- 特点: 拥有未觉醒完全的异能潜力

**自定义选项**:
- 外观: 发型、发色、瞳色、服装 (深度定制)
- 声音: 多声优选择
- 背景故事: 影响起始属性和初始关系

### 4.2 成长系统
**属性**:
- 体能 (生命值、耐力)
- 敏捷 (移动速度、闪避)
- 技巧 (武器精准度、黑客能力)
- 魅力 (对话选项、交易价格)
- 异能 (能量条、技能强度)

**技能树**:
- 战斗分支: 武器专精、格斗技巧、战术能力
- 异能分支: 时间、念力、元素、幻影、防御
- 社交分支: 谈判、恐吓、魅力、领导力
- 技术分支: 黑客、工程、驾驶、医疗

### 4.3 重要NPC
| 角色 | 身份 | 作用 |
|------|------|------|
| 绫波零 (Rei) | 神秘黑客 | 任务发布、技术支持、可恋爱 |
| 龙崎 (Ryuji) | 前红龙会干部 | 战斗导师、势力任务中介 |
| 艾琳娜博士 | 天网科学家 | 异能升级、主线剧情关键 |
| 老陈 | 修车店老板 | 载具改装、情报交易 |
| 幽灵 | 传说佣兵 | 高难度挑战、终极装备 |

## 5. 经济与装备

### 5.1 货币系统
- **现金**: 基础货币，任务奖励、拾取
- **加密货币**: 黑市交易、黑客任务
- **势力声望点**: 各势力专属，兑换特殊物品

### 5.2 装备类型
**武器**:
- 近战: 拳头、警棍、武士刀、电锯
- 手枪: 普通、消音、全自动
- 步枪: 突击、狙击、霰弹
- 特殊: 火箭筒、异能增幅器、实验武器

**防具**:
- 服装: 风格外观 + 轻微防护
- 护甲: 战术背心、动力装甲 (影响移动)
- 配件: 头盔、护目镜、背包

**道具**:
- 医疗: 绷带、医疗包、异能恢复剂
- 战术: 手雷、烟雾弹、EMP、诱饵
- 黑客: 破解工具、病毒U盘、干扰器

## 6. 动漫视觉风格

### 6.1 渲染技术
- **Cel-Shading**: 卡通着色 + 描边效果
- **后处理**: 
  - 动态线条 (速度线、冲击线)
  - 漫画分镜效果 (暴击时)
  - 表情符号弹出 (情绪表达)
- **光影**: 高对比度 + 色彩分级 (不同区域不同色调)

### 6.2 UI 设计
- 极简 HUD，信息显示通过世界空间 UI
- 对话采用视觉小说风格 (立绘 + 对话框)
- 菜单采用科技感设计 (全息投影风格)

### 6.3 动画风格
- 夸张的动作表现 (弹簧物理、变形)
- 表情丰富 ( anime 风格表情)
- 战斗特效华丽 (粒子效果 + 手绘风格贴图)

## 7. 音频设计

### 7.1 音乐
- **探索**: 电子音乐 + 日系摇滚，根据区域变化
- **战斗**: 高强度 J-Rock / 电子核
- **剧情**: 管弦乐 + 钢琴，情感渲染
- **载具**: 根据电台选择 (多种风格)

### 7.2 音效
- 武器: 夸张化枪声 (动漫风格)
- 异能: 合成器音效 + 环境混响
- UI: 科技感提示音
- 环境: 城市氛围 (根据区域变化)

### 7.3 语音
- 主线全语音 (日语/中文/英语可选)
- 知名声优配音
- 路人NPC使用AI生成语音 (多语言)

## 8. 技术实现

### 8.1 UE5 特性使用
- **Nanite**: 城市建筑、载具高模直接导入
- **Lumen**: 全局光照，霓虹灯效果
- **Chaos**: 载具物理、建筑破坏
- **MetaHuman**: 高质量角色基础
- **Control Rig**: 动态动画调整
- **Niagara**: 异能特效、环境效果

| PC (Mid) | 1440p | 60fps | High |
